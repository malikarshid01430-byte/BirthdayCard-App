# BirthdayCard App 🎉

A simple Android application that displays a customizable birthday greeting card.

## 🚀 Features
- Birthday greeting UI
- Customizable text
- Clean and simple design

## 🛠️ Tech Stack
- Kotlin
- Android Studio
- XML Layouts

## ▶️ How to Run
1. Clone the repository
2. Open in Android Studio
3. Run the app
